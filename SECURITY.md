# Security Policy

Please do not open public issues for security vulnerabilities. Instead, report them privately using the instructions below.

## Reporting a Vulnerability

If you discover a security issue, please email the project maintainer directly. Include as much detail as possible, including a proof of concept if you have one.

I will acknowledge your report within 48 hours and provide a timeline for a fix. Please wait for a coordinated disclosure before sharing details publicly.

## Supported Versions

Only the latest release of MicroGAN is actively supported for security updates.
